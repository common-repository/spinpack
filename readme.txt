اسپینپک بهترین و کاربردی ترین افزونه پشتیبانی برای سایت های وردپرسی میباشد.

== Description ==

اسپینپک یک پلتفرم پشتیبانی برای وردپرس میباشد که یک دکمه شناور توی صفحات سایت شما اضافه میکند که میتوانید به آن دکمه لینک بدید و به عنوان لینک پشتیبانی سایت استفاده کنید و از فروشتان لذت ببرید.

افزونه اسپینپک ساده ترین و زیبا ترین پنل تنظیمات را دارد که خیلی راحت میتوانید آیتم های افزونه را سفارشی سازی یا تنظیم کنید.

اگر مشکلی در رابطه با استفاده یا هر چیز دیگه ای برای این افزونه دارید میتوانید در تلگرام با آیدی `@venoustarh_support` تماس بگیرید و مشکلتون رو بگید تا رفع کنیم.


== Installation ==

برای نصب افزونه از این مراحل استفاده کنید.

1. فایل های افزونه را در دایرکتوری `/wp-content/plugins` آپلود کنید، یا افزونه را مستقیماً از طریق صفحه افزونه های وردپرس نصب کنید.
2. افزونه را از طریق صفحه `Plugins` در وردپرس فعال کنید.
3. بعد از فعال کردن افزونه به بخش تنظیمات افزونه بروید و تنظیمات مورد نظر خود را اعمال کنید و ذخیره کنید.
4. سپس می توانید ویجت `spinpack` را در صفحات وب خود مشاهده کنید.
